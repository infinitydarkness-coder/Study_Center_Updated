### 🔧 Installation Steps

#### **Step 1️⃣ Clone Repository**
```bash
git clone https://github.com/yourusername/study-hub.git
cd study-hub-updated
```

#### **Step 2️⃣ Install Dependencies**
```bash
# Using npm (recommended)
npm install

# Or using alternative package managers
yarn install
pnpm install
```

#### **Step 3️⃣ Configure Environment**
```bash
# Create environment file
cp .env.example .env.local

# Add your Supabase credentials
cat >> .env.local << 'EOF'
VITE_SUPABASE_URL=https://your-project.supabase.co
VITE_SUPABASE_ANON_KEY=your-anon-key-here
EOF
```

#### **Step 4️⃣ Launch Development Server**
```bash
npm run dev
```

🌐 **Open your browser**: http://localhost:5173

```typescript
import { supabase } 
  from '@/lib/supabase'

const signIn = async (
  email: string,
  password: string
) => {
  const { data, error } = await 
    supabase.auth
      .signInWithPassword({
        email,
        password,
      })
  return { data, error }
}
```

**📊 Data Operations**

```typescript
// Fetch data
const getUsers = async () => {
  const { data, error } = 
    await supabase
      .from('users')
      .select('*')
  return { data, error }
}

// Real-time subscription
supabase
  .channel('updates')
  .on('postgres_changes',
    { event: '*', 
      schema: 'public' },
    (payload) => {
      console.log(payload.new)
    }
  )
  .subscribe()
```
